# AWM Global — Corporate Presentation 2026

Presentación corporativa profesional para **AWM Global**, una firma de wealth management especializada en productos financieros estructurados y gestión patrimonial internacional.

## Características

- **27 diapositivas** en formato A4 horizontal (landscape)
- Identidad visual AWM completa con paleta de verdes corporativos
- Tipografía Montserrat en todas sus variantes
- Imágenes de stock profesionales de Unsplash
- Generación de PDF descargable (A4 landscape, 297×210mm)
- Navegación por teclado (← →) y panel de miniaturas
- Contenido exclusivamente basado en [awmglobal.net](https://www.awmglobal.net)

## Stack Tecnológico

- **React 19** + TypeScript
- **Vite** como bundler
- **Tailwind CSS v3** con paleta AWM personalizada
- **jsPDF** + **html2canvas** para generación de PDF
- **Montserrat** via Google Fonts

## Instalación y Uso

```bash
# Instalar dependencias
cd awm-presentation
npm install

# Desarrollo local
npm run dev

# Build de producción
npm run build

# Preview del build
npm run preview
```

## Generación de PDF

1. Abrir la aplicación en el navegador
2. Hacer clic en el botón **"Descargar PDF"** (verde, esquina superior derecha)
3. Esperar 1-2 minutos mientras se procesan las 27 diapositivas
4. El archivo `AWM_Global_Presentation_2026.pdf` se descargará automáticamente
